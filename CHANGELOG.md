## v1.3 · 2026-06-11 · 大体重 BMI 分档 + 有氧/1RM 两个小算盘

- **大体重起始配额下调(D13,减脂期自动)**:BMI>28 碳水起始上限压到 男2.5/女2.1 g/kg,BMI>32 压到 男2.0/女1.7(下限=上限−0.3;该档休息日碳水 −0.3 而非 −0.5,贴合女版配额表大体重行)。按 profile 身高自动分档,减下来跨档自动回升;无身高/增肌/维持不受影响。依据:松松"大体重低配额"(单位体重基代 1.3–1.4 倍差、活动消耗 1.5 倍差)。诚实标注:女 BMI>32 档比女版配额表训练日行(1.8–2.1)略保守,符合松松"算出来偏低是故意的"原则,饿得受不了可按表上调——已写进 references。
- **有氧心率消耗计算器** `tracker.py cardio`:松松 Excel sheet16 心率法(活动心率÷静息心率×6.4−6.2,每kg每小时),大体重 ≥90kg 按原表衰减(0.98→0.90);支持算单次消耗、"每周 N 小时→每天能多吃多少"(自动÷7)+ 食物换算;范围外输入有提示。注:逐格核对 Excel 确认衰减只随体重、与心率无关;原表 115kg 列 0.92 疑笔误,按单调 0.90 处理。
- **1RM 最大力量预测** `tracker.py 1rm`:Excel sheet24 九公式(Adams/Brown/Brzycki/Lander/Lombardi/Mayhew/O'Connor/Wathen/Welday)全部逐格誊录,回传逐公式估值+中位数+按性别推荐(女:Brown/Brzycki/Lander,男:Lombardi,系 Excel 原标注);附松松训练容量参考(周频率/分化/大9小6组数/6–12RM)。次数>12 提示误差大。
- 两个小算盘**只算不存档**;公式已全部写入 `references/松松减脂框架.md` §13.5 供查证。

## v1.2 · 2026-06-11 · 逐字精读松松全部视频+Excel,补全没消化透的方法论

把 12 篇视频逐字稿 + Excel 套表逐字过了一遍,对照引擎补漏:
- **脂肪改成区间**(下限–目标–上限):下限防"吃太少→性激素/经期紊乱",上限防超标——三大营养素统一成区间。
- **脂肪刺客提醒**:看板点评会在记到 五花/排骨/坚果/油条/糖油混合物 等时提醒"看不见的脂肪"易低估(宽油炒蛋20–30g、两把坚果40g)。
- **瘦肉判定 / 高脂肉黑名单**:皮40%、肥肉/排骨/五花90%、裹粉油炸、肉丸肉肠≠瘦肉(写进 references + SKILL)。
- **平台期 9 查逐条 + 铁律**:连续 1–2 周不动才算平台,先逐条查执行(定量/拆混合菜/隐形脂肪/有氧/水果扣主食/外食/酒/特殊),确认无缺口才每天−150kcal 或周+1000kcal 有氧。
- **进食顺序**:先吃菜/蛋白后吃主食,胰岛素差 30%。
- **减脂终点**:接近 女 BMI≈20–21 / 男≈22–23 时提醒别追更低、可转增肌。
- **有氧体重分档**:≥80kg 靠吃、≤70kg 每周2小时、70–80看饿,上限<4h/周、放力量之后。
- **调整尺度**:碳水 ±0.5g/kg(≈±100–200kcal)、只减碳加蛋、地板 2g/kg;体重降 5–10kg 才重算配额;肌酸首月吸水 1–2kg 要扣。
- **知识库**:补固定重量食物表、酒精换算(1两白酒=200kcal=50g碳水)、有氧换饮食。

## v1.1 · 2026-06-11 · 补上「日内碳水分配」

- **修复**:此前引擎只看全天碳水总量,不看**单顿占比**,导致"一顿吃掉全天 ~50% 碳水"也判为没问题。
- 现按松松第 5 节「日内分配」补全:**训练日**碳水主力放练后餐(40–50%)、早饭~20–30%、练前~20%;**休息日**无练后窗口→碳水摊匀到三餐(每顿别超约 1/3)、先吃菜/蛋白后吃主食。
- 看板饮食点评 + SKILL 判断口径都会在**单顿碳水占全天 ≥40%**(尤其休息日非练后餐)时提醒"别堆、往练后挪/摊匀",哪怕全天总量没超。

## v1.0 · 2026-06-10 · 首个版本 🎉

**核心能力上线：**
- **建档**：填写个人资料；历史体重支持三选一（苹果健康导入 / 截图识别 / 直接跳过从今天起）。
- **大白话记录**：体重、饮食、运动、睡眠、经期 → 自动结构化；混合菜拆原料；营养表优先、否则按松松营养率估。只记已发生、不记计划。
- **松松方法论引擎**：按 性别 × 阶段（减脂/增肌/维持）× 训练日/休息日 给碳蛋脂**配额区间**（下限–目标–上限）；7 日均 + 周均看趋势；速度评估、平台期排查、降太猛提醒；陪跑点评 + 基于缺口的下一餐具体建议。
- **本地可视化看板**：单 HTML，含里程碑/今日状态/体重趋势/饮食配额/活动日历/身体成分/今日夸夸/本周小结；每次记录自动刷新；组件化统一风格；纯本地、无公开网页。
- **漏记追问**：聊天内主动追问当天缺记；可选每晚定时提醒（需 Claude Code 定时任务）。
- **食物库双层**：松松营养率为公用层；用户给了营养的食物自动存入个人常吃库并提示。
- **内置教程 + PRD + 本更新日志**：安装即了解全部能力。
