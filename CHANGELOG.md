## v1.1 · 2026-06-11 · 补上「日内碳水分配」

- **修复**:此前引擎只看全天碳水总量,不看**单顿占比**,导致"一顿吃掉全天 ~50% 碳水"也判为没问题。
- 现按松松第 5 节「日内分配」补全:**训练日**碳水主力放练后餐(40–50%)、早饭~20–30%、练前~20%;**休息日**无练后窗口→碳水摊匀到三餐(每顿别超约 1/3)、先吃菜/蛋白后吃主食。
- 看板饮食点评 + SKILL 判断口径都会在**单顿碳水占全天 ≥40%**(尤其休息日非练后餐)时提醒"别堆、往练后挪/摊匀",哪怕全天总量没超。

## v1.0 · 2026-06-10 · 首个版本 🎉

**核心能力上线：**
- **建档**：填写个人资料；历史体重支持三选一（苹果健康导入 / 截图识别 / 直接跳过从今天起）。
- **大白话记录**：体重、饮食、运动、睡眠、经期 → 自动结构化；混合菜拆原料；营养表优先、否则按松松营养率估。只记已发生、不记计划。
- **松松方法论引擎**：按 性别 × 阶段（减脂/增肌/维持）× 训练日/休息日 给碳蛋脂**配额区间**（下限–目标–上限）；7 日均 + 周均看趋势；速度评估、平台期排查、降太猛提醒；陪跑点评 + 基于缺口的下一餐具体建议。
- **本地可视化看板**：单 HTML，含里程碑/今日状态/体重趋势/饮食配额/活动日历/身体成分/今日夸夸/本周小结；每次记录自动刷新；组件化统一风格；纯本地、无公开网页。
- **漏记追问**：聊天内主动追问当天缺记；可选每晚定时提醒（需 Claude Code 定时任务）。
- **食物库双层**：松松营养率为公用层；用户给了营养的食物自动存入个人常吃库并提示。
- **内置教程 + PRD + 本更新日志**：安装即了解全部能力。
