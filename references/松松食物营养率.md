# 松松「日常食物营养率」表(种子数据)

> 来源:好人松松《健身Excel超级套表》Sheet `19日常食物营养率`,原样抽取。
> **读法**:碳水率/蛋白质率 = 每克食物含该宏量的克数(0.75 = 每 100g 含 75g)。带「g/个、g/盒」的为绝对含量,导入时单独处理。
> **注意**:原表大多只给营养率 + GI,**无热量/脂肪率列**;热量需用 碳×4 + 蛋×4 + 脂×9 反算。脂肪走「行为规则」不逐项计。
